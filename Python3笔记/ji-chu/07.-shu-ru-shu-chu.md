# 07.输入输出

## 输入

`input()` 函数

用法：

```text
a = input("提示语句")
print(a)
```

## 输出

`print()` 函数，可以传多个参数，还可以通过设置 `sep` 、`end` 参数的值来改变间隔符和换行  
`sep` 参数控制用什么作为要打印参数的间隔，默认为空格  
`end` 参数设置用什么作为该次打印的结尾，默认为回车 `\n`  
两个参数值必须是字符串

具体用法：

```text
# 不换行
print("1", '2', sep='', end='')
print(1, 3, sep='\n', end='end')
```
