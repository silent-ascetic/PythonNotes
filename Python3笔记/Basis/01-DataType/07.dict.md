# 07.Dict

字典是以键值对的形式存储元素的可迭代数据类型，类似数据结构中的哈希表  
字典中的key值不可以重复，value值可以。其中key只能是不可变数据类型，value随意

## 定义格式

```python
# 定义空字典
a = {}
# 定义常规字典
b = {1:'3'}
# 1为key，'3'为value
```

## 增删查改

### 插入元素

```python
a = {1:'1'}
# a[key] = value
a[2] = '2'
```

### 删除元素

```python
a = {1:'1', '2':2}
del a['2']
```

### 查找元素

根据key查找value

```text
a = {1:'1', '2':2}
# key不在字典中会报错
print(a['2'])
# key不在字典中不会报错
print(a.get('2'))
```

### 更改value值

```python
a = {1:'1', '2':2}
# key必须存在于字典中，否则是插入
a[1] = 1
```
