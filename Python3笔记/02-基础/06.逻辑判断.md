# 逻辑判断

python中判断语句只有if（和while循环），不支持switch

判断相等或不等用 `==` 和 `!=`, 用 `None` 表示空


## 与或非

### 逻辑与（and）

相当于C语言中的 `&&`  
举个栗子：
```eval-python
print(True and False)
```

### 逻辑或（or）

相当于C语言中的 `||`  
举个栗子:
```eval-python
print(True or False)
```

### 逻辑非（not）

类似C语言中的 `!`,not后面可以接表示真假的表达式或变量表示非真或非假，但不能与等号连用
例如：
```eval-python
print(not 1 == 3)
```
